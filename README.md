# Excursion

## Description
This project is a promotional webpage that advertises a new app called Excursion.  It was built as an off-platform project for Codecademy to practice creating a clean, centered web layout using HTML5 and CSS3.
## Features
- Clean, centered layout matching the design mockup
- Looping background video
- Links to download the app
